### var

- 변수의 중복 선언이 허용된다.
- 함수 레벨 스코프를 지원한다. (함수의 코드 블록만을 지역 스코프로 인정한다. if문, for문의 코드 블록 내에 선언해도 전역 변수가 된다.)
- 변수 호이스팅이 발생한다. (변수 선언문이 선두로 끌어 올려진 것처럼 동작한다.)

### let, const

- 변수 중복 선언이 금지된다.
- 블록 레벨 스코프를 지원한다. (모든 코드 블록 (함수, if문, for문, while문, try/catch문)을 지역 스코프로 인정한다.)
- 변수 호이스팅이 발생하지 않는 것처럼 동작한다. (let 키워드로 선언한 변수는 "선언 단계"와 "초기화 단계"가 분리되어 실행된다.)
- 전역 객체의 프로퍼티가 되지 않는다. (var 키워드로 선언한 전역 변수와 전역 함수, 선언하지 않은 변수에 값을 할당한 암묵적 전역은 전역 객체 window의 프로퍼티가 된다.)

### const

- const 키워드로 선언한 변수는 반드시 선언과 동시에 초기화를 해야한다.
- 재할당이 금지된다.
